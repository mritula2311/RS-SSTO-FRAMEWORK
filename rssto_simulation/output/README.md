# Output Visualization Images

This folder contains all visualization outputs generated by the RS-SSTO evacuation simulation framework.

## Files

- **behaviour_comparison.png** - Comprehensive visual comparison showing the evacuation behavior of all five algorithms (SFM, PSO, ACO, APF, RS-SSTO)
- **comparison_overview.png** - Side-by-side comparison of all five algorithms with their final states
- **efficiency.png** - Bar chart comparing evacuation efficiency (percentage of agents evacuated)
- **evacuated.png** - Bar chart showing the number of agents successfully evacuated per algorithm
- **evacuation_time.png** - Bar chart comparing total evacuation time in frames
- **throughput.png** - Bar chart comparing throughput (evacuated agents per frame)

## Description

The behaviour_comparison.png is the key visualization that demonstrates how each algorithm handles the evacuation scenario, showing:
- Agent movement patterns
- Obstacle avoidance strategies
- Exit convergence behavior
- Congestion handling

All images are generated using Matplotlib and represent the simulation results under identical initial conditions for fair comparison.
